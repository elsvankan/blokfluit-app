# Blokfluit Buddies (MVP)

Een Nederlandstalige, mobile-first web-app (Next.js + TypeScript + Tailwind) om kinderen stap voor stap blokfluit te leren oefenen.

## Wat de app doet
- **Noten leren** met flashcards (nootnaam + vingerzetting)
- **Patronen oefenen** met actieve noten
- **Dagelijkse oefening** in kleine stappen
- **Voortgang bijhouden** met sterren, streaks en lastige noten
- **Adaptieve leerlogica**: lastige noten komen vaker terug

## Starten (lokaal)
1. Installeer dependencies:
   ```bash
   npm install
   ```
2. Start de ontwikkelserver:
   ```bash
   npm run dev
   ```
3. Open: http://localhost:3000

## Deployen op Vercel
1. Push deze repository naar GitHub.
2. Ga naar vercel.com en log in.
3. Klik **Add New... → Project**.
4. Selecteer je GitHub-repo.
5. Klik **Deploy**.
